Letakkan semua gambar Anda (logo.png, project-1.jpg, dll) di folder ini (public).
Kemudian ubah nama file di dalam objek CONFIG pada src/App.tsx agar sesuai dengan nama file yang Anda letakkan di sini.

Contoh:
Jika Anda meletakkan gambar logo dengan nama "logo-ditz.png" ke dalam folder ini, maka di dalam src/App.tsx Anda ubah:
logo: "/logo-ditz.png"
